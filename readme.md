## Compatibility
Compatible with Laravel v5.3.21. Not tested below Laravel v5.3.21

## Install
Composer require

	composer create-project gurinder/laravel-plus-admin-theme FolderName
  	
Update Node

	npm update
	  	
Update Composer

	composer update
	
Run Gulp Setup

	gulp setup
